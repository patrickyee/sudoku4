# 4x4 Sudoku

A simple 4x4 Sudoku game built with Vue.js.

## How to Use

Simply open `index.html` in your browser to start playing.

## Features

- Random 4x4 Sudoku puzzle generation
- Real-time input validation
- Timer and best time tracking
- Undo functionality
- Optional color hints
- Mobile-friendly interface

## Deployment

No compilation needed. Just upload `index.html` to your CDN or web server.
